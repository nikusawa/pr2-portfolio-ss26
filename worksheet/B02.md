## 1. Git-Spiel

### 1.1
Auf `master` die Datei `hero.md` editieren (z. B. Leerzeile hinzufügen) und committen:
    
    echo "" >> hero.md
    git add hero.md
    git commit -m "Neue Leerzeile in hero.md auf master"

Anschließend end in master mergen:

    git merge origin/end

### 1.2

`questlog.md` auf `master` ändern (z. B. einen Satz am Anfang des letzten Absatzes einfügen):

    sed -i '1s/^/Neuer Satz auf master. /' questlog.md
    git add questlog.md
    git commit -m "Änderung am Anfang von questlog.md auf master"

`sed` command gibt es nur auf Linux.
